MathML: Einstein’s Formula

E=mc2

Why wasn’t MathML widely usable until recently?

web browsers didn’t properly support it for a long time. For many years, only Firefox could display MathML correctly, while Chrome, Safari, and Edge either ignored it or showed it incorrectly. Rendering math on the web is also harder than regular text because it involves complex layouts for fractions, roots, and equations, which made browser developers avoid implementing it. Since it didn’t work consistently, most websites used tools like MathJax or just showed math as images instead. This created a cycle where no one used MathML because it wasn’t reliable. Only recently, with the creation of the new MathML Core standard in 2023 and full support added to Chrome, Safari, and Firefox, did it finally become consistent, fast, and ready for real use across the web.
